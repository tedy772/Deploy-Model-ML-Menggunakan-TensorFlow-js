# Deploy-Model-ML-Menggunakan-TensorFlow-js
Proyek Analisis Sentimen pada Review Sebuah Restoran, Tujuan dari proyek ini adalah untuk membuat sebuah model yang dapat memprediksi apakah review sebuah restoran merupakan review positif atau negatif. Proyek ini menggunakan dataset yield dari Kaggle, dataset tersebut yang berisi kumpulan review terhap sebuah restoran.
